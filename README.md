# Node

A beginner-friendly Node.js project that demonstrates the use of core Node modules like `fs`, `os`, `http` and simple JavaScript functions.  
This repository includes example scripts that help you practice how Node works without external frameworks. 

## Project Overview

This repository contains several simple Node.js scripts that demonstrate basic Node concepts — such as:

✔️ Using file system operations (`fs`)  
✔️ Getting operating system info (`os`)  
✔️ Creating a basic HTTP server  
✔️ Writing simple JavaScript functions

## Files Description

File --> Purpose 
-----------------
`Fs_module.js` --> Demonstrates reading, writing, and working with files using Node’s built-in `fs` module. 
`Os_module.js` --> Shows how to get system information using the `os` module (like CPU, memory, platform). 
`http_module.js` --> Creates a simple Node.js server using the `http` core module. 
`add.js` --> Example of a basic JavaScript function to add numbers — useful for logic practice.
